Quick Start

Get up and running in less than 30 seconds. The AI Builder API is designed to be simple and accessible.

1. Make a Request

Send a POST request to /api/chat. No headers are required.

JSON Request 
POST /api/chat
{
  "model": "z-ai/glm4.7",
  "messages": [
    {
      "role": "user",
      "content": "Hello"
    }
  ],
  "stream": false
}

2. Receive Response

The API returns a standard JSON object containing the AI's reply.

JSON Response 
{
  "choices": [
    {
      "message": {
        "role": "assistant",
        "content": "Hello! How can I help you today?"
      }
    }
  ]
}

Chat API Reference

The core endpoint for interacting with the AI models.

Endpoint

POST /api/chat

Request Body

Field Type Required Description
model string Yes Model ID (e.g., z-ai/glm4.7)
messages array Yes List of message objects with role and content.
stream boolean No Enable SSE streaming (default: false).
mode string No Behavior mode: chat, code, reasoning.

Streaming Guide

Streaming allows you to display partial responses as they are generated, significantly improving the perceived latency of your application.

How it works

When stream is set to true, the API sends Server-Sent Events (SSE). Each chunk is a JSON object prefixed with data:.

Stream Format
data: {"choices":[{"delta":{"content":"Hello"}}]}
data: {"choices":[{"delta":{"content":" world"}}]}
data: [DONE]

JavaScript Implementation

Use the native fetch API and getReader() to consume the stream.

streaming.js 
async function streamChat() {
  const response = await fetch('http://aibuilder.eu.cc/api/chat', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      model: "z-ai/glm4.7",
      messages: [{ role: "user", content: "Tell me a joke" }],
      stream: true
    })
  });

  const reader = response.body.getReader();
  const decoder = new TextDecoder();

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    
    const chunk = decoder.decode(value);
    const lines = chunk.split('\n');
    
    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const json = JSON.parse(line.slice(6));
        if (json === "[DONE]") return;
        
        const content = json.choices[0].delta.content;
        console.log(content); // Update UI here
      }
    }
  }
}

AI Modes System

Customize the AI's behavior by using the mode parameter. This changes the system prompt and output formatting.

💬 Chat

Default conversational assistant. Helpful, creative, and casual.

"mode": "chat"

🧠 Reasoning

Shows step-by-step thinking before the final answer.

"mode": "reasoning"

🧑‍💻 Code

Optimized for programming. Returns markdown code blocks with syntax highlighting.

"mode": "code"

🎮 Scratch

Generates XML-based Scratch blocks for educational visual coding.

"mode": "scratch"

📄 Docs

Structured, formal responses suitable for documentation generation.

"mode": "docs"

Error Codes

Standard HTTP status codes used by the API.

Code Description
400 Invalid Request (Malformed JSON or missing fields).
404 Endpoint not found.
429 Rate limit exceeded (Free tier limits apply).
500 Internal server error. Please try again later.
